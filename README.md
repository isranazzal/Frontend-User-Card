# User Profile Card

## Description

A frontend project that demonstrates the creation of a modern user profile card using **HTML5** and **CSS3**. The design includes a profile image, user information, location, biography, and an edit profile section.

This project focuses on building reusable UI components while practicing layout, spacing, typography, and styling.

---

## Technologies Used

- HTML5
- CSS3

---

## Features

- Modern profile card layout
- User avatar
- Location section with icon
- User biography
- Edit profile link
- Clean and organized CSS styling
